Course syllabus

I have designed the bootcamp syllabus to not just build you as a good Frontend Developer, but make you ready for the professional world.

  • Browser & Internet

    We'll understand how a Browser enables us to use the Web and how we can use the inbuilt tools to get insights into how any website works.

  • GIT

    We'll learn the basics of Git and how to use it to manage our code.

  • HTML

    We'll learn how HTML enables communication through content and how we can structure our content so that its accessible to everyone.

  • CSS

    We'll develop clear mental models of how this seemingly daunting language works and how to approach learning and using it in the right manner.

  • JavaScript

    We'll focus on building a strong foundation of JavaScript and structuring code for web applications. In the end, we'll also touch a bit on React JS framework.

  • Web Accessibility & Performance

    This won't be a separate module, but we'll touch it throughout the camp.

  • Design for Engineers

    I believe Engineers should a bit of visual design too. It helps appreciate the work of designers and also developers an empathy for your customers. This won't be a separate module, but we'll touch it throughout the camp.

  • Working with AI

    AI has become a need of the hour today. We'll learn how to use AI to enhance our productivity, while making sure we don't loose that human taste and become an AI puppet. "Vibe in the code, not to the end user".

  • Building end-to-end products

    Though we'll focus on becoming a cracked Frontend Developer, knowing how to build end-to-end products is an important skill, more so in current times. We'll learn how to build simple end-to-end products as a Frontend Developer.

  • Professional Etiquettes

    Programming applications is not enough to get hired by a good team.Soft skills are equally important. We'll learn how to be a good team player.
